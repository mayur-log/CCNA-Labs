# Day 03 - OSI Model Lab

## Objective
Learn the OSI Model and observe how packets travel through a network using Cisco Packet Tracer Simulation Mode.

## Topology
- 1 Server (SRV1)
- 1 PC (PC1)
- 2 Cisco 2960 Switches (SW1, SW2)
- 2 Cisco 2911 Routers (R1, R2)

## Network Addressing
| Device | IP Address |
|---------|------------|
| SRV1 | 192.168.1.100/24 |
| R1 G0/0 | 192.168.1.1/24 |
| R1 G0/1 | 10.0.0.1/24 |
| R2 G0/0 | 10.0.0.2/24 |

## Protocols Observed
- ARP
- DHCP
- ICMP
- OSPF

## Concepts Learned
- OSI Model (7 Layers)
- Packet Flow
- ARP Resolution
- DHCP Process
- ICMP (Ping)
- OSPF Routing
- Simulation Mode Analysis
